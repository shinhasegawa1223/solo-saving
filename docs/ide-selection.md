# IDE選定：Google Antigravityの採用理由

本プロジェクトでは開発環境として**Google Antigravity**を採用しました。
選定プロセスにおいて、Cloud Code、Kiro、Google Antigravityの3つ比較検討を行いました。

## 選定候補

1.  **Cloud Code**: Google Cloud開発に特化したIDEプラグイン。
2.  **Kiro**: AWSが開発中の実験的なAgentic IDE。
3.  **Google Antigravity**: Google DeepMindによる先進的なAgentic AIコーディングアシスタント。

## 選定理由：コストパフォーマンスと機能

最終的に**Google Antigravity**を選定した最大の理由は、高い機能性を持ちながら非常に安価なコストで利用できる点です。

### 料金体系とキャンペーン

Google Antigravityは現在、非常に魅力的なキャンペーンを実施しており、コスト面で他の選択肢を圧倒しています。

### 料金・クレジット詳細（最初の半年間）

| 項目 | 期間・内容 | 概算費用 | 備考 |
| :--- | :--- | :--- | :--- |
| **初期導入** | 最初の3ヶ月 | **約 950円** | キャンペーン価格。通常より大幅割引。 |
| **継続利用** | 4ヶ月目〜6ヶ月目 | 別途プラン | 今後のプラン体系に準拠。 |

各ツールの「最も安価、かつ実用的なプラン」での比較です。

| ツール | 対象プラン | 月額費用（概算） | 備考 |

> [!TIP]
> **現在キャンペーン実施中！**
> Google Antigravityは現在、最初の3ヶ月間を合計**950円**（月額換算 約317円）で利用できるキャンペーンを実施しており、初期導入コストを劇的に抑えることが可能です。

Google Antigravityは、通常料金でも他社のProプランと同等水準でありながら、強力なAgentic AI機能（自律的なタスク実行）を備えているため、非常に高いコストパフォーマンスを誇ります。

### 機能・柔軟性比較

コスト以外の技術的な特徴の比較です。

| 比較項目 | **Google Antigravity** | **Cloud Code (Gemini)** | **Kiro (AWS)** |
| :--- | :--- | :--- | :--- |
| **主な形態** | **独立したAgentic開発環境** | IDEプラグイン (VS Code等) | スタンドアロン型IDE (VS Codeベース) |
| **自律性 (Agentic)** | **極めて高い** (Terminal/Browser連携) | 中程度 (Chat/Completion中心) | 高い (Spec駆動・自律修正) |
| **マルチファイル編集** | **強力** (全体像を把握して一括修正) | 制限あり (ファイル単体中心) | 強力 (計画に基づいて複数編集) |
| **対応モデル** | **Gemini, Claude, GPT等 選択可能** | Gemini 固定 | Claude (Bedrock経由) 中心 |
| **強み** | **タスクの「丸投げ」が可能** | Google Cloudインフラ連携 | 仕様書(Spec)ベースの堅実な開発 |

> [!IMPORTANT]
> **Antigravityの決定的な違い**
> Antigravityは、単なるコード補完にとどまらず、**「ターミナルの実行結果を見てエラーを直す」「ブラウザでUIを確認しながらデバッグする」**といった人間のような試行錯誤をAIが自律的に行える点が最大の特徴です。

### 結論：なぜGoogle Antigravityか？

1.  **初期コストの圧倒的低さ**: 最初の半年間をキャンペーンと通常料金の組み合わせで非常に安価に運用できる。
2.  **真のAgentic体験**: ターミナルやブラウザを操作できるAIにより、記述量自体を大幅に削減できる。
3.  **モデルの柔軟性**: 状況に応じて最適なモデルを使い分けることができる。

以上の理由から、本プロジェクトでは最速・最小限の労力で高機能なアプリを構築するためにGoogle Antigravityを採択しました。
